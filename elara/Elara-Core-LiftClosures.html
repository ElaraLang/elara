liftClosures :: forall (r :: [Effect]). LiftClosures r => Expr Var -> Eff r (Expr Var) Source #

Lift closures in a Core expression. This turns all closures into top-level functions. For example: let add5 x = let f y = x + y in f 5 becomes: let add5_f x y = x + y let add5 x = add5_f x 5
