tokenEndsExpr :: Token -> Bool Source #

Returns True if the token ends an expression. This is used to determine whether line breaks should be treated as semicolons If the token doesn't end an expression, then a line break is skipped. If the token does end an expression, then a line break is treated as a semicolon.
