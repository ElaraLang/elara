data DiagnosticWriter t (a :: Type -> Type) b where Source #

Essentially a very specialised Writer effect for Diagnostics, but safer | Because the Semigroup instance for Diagnostics is a little funky, there's a chance of accidentally overwriting the | entire FileMap, so this effect is designed to prevent that, and also prevent the state / writer boilerplate as an added bonus :)
