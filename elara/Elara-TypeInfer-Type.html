data AxiomScheme (loc :: k) Source #

An axiom scheme QQ

Constructors

EmptyAxiomScheme

The empty axiom scheme 𝜖

ConjunctionAxiomScheme (AxiomScheme loc) (AxiomScheme loc)

The conjunction of two axiom schemes, QQ₁ ∧ QQ₂

ForallAxiomScheme

A universal quantification of an axiom scheme, ∀α. QQ ⇒ QQ Practically this could be a declaration like forall a. Eq a => Eq [a]

    Type variable to be quantified

    Constraint for the quantified variable
